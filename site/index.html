Keeping it fresh

Run python run_daily.py each morning (or schedule run_daily.bat with Task Scheduler). It grades yesterday's games, retrains on every result so far, refreshes scores, leaders, and today's predictions, and this page updates automatically on refresh. The stamp in the header shows the date of the last model run — if it isn't today's date, the feed is stale.
